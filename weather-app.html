Weather App Challenge

Welcome to the Weather App challenge! Follow the instructions below to complete the challenge:

  1. Create an input field where users can type a city name.
  2. Add a button to fetch weather data for the entered city.
  3. Display the weather information (e.g., temperature, humidity, etc.) on the page.
  4. Use a weather API like OpenWeatherMap to fetch the data.

Good luck!
